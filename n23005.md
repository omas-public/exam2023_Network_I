## 問題 01 [curl]

https://www.codingbat.com/login に DATA {"uname":"omas","pass","hogehoge"} curl を用いてリクエストするコマンドを記述せよ

解答欄

```bash
# この下にコマンドを記述
DATA="{uname":"omas","pass","hogehoge"}
curl -isSc https://www.codingbat.com/login -d "DATA"

```

## 問題 04 [netcat]

netcat(nc)を用いて echo サーバー及び echo クライアントを作成するコマンドを記述せよ
但し，Host は localhost,　 port 番号は 7777 とする

解答欄(サーバー側)

```bash
# この下にコマンドを記述
nc -l 7777

```

解答欄(クライアント側)

```bash
# この下にコマンドを記述
nc localhost 7777

```

## 問題 05 [netcat]

netcat(nc)を用いてクライアントからファイルを転送するコマンドを記述せよ
ファイル名はクライアント側は shadow サーバー側は sendfile とする

解答欄(サーバー側)

```bash
# この下にコマンドを記述
nc -nlvp sendfile

```

解答欄(クライアント側)

```bash
# この下にコマンドを記述
nc -nlvp shadow

```
