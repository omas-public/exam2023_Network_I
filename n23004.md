## 問題 04 [netcat]

netcat(nc)を用いて echo サーバー及び echo クライアントを作成するコマンドを記述せよ
但し，Host は localhost,　 port 番号は 7777 とする

解答欄(サーバー側)

```bash
# この下にコマンドを記述
nc -l -p 7777

```

解答欄(クライアント側)

```bash
# この下にコマンドを記述
nc 127.0.0.1 7777

```

## 問題 05 [netcat]

netcat(nc)を用いてクライアントからファイルを転送するコマンドを記述せよ
ファイル名はクライアント側は shadow サーバー側は sendfile とする

解答欄(サーバー側)

```bash
# この下にコマンドを記述
nc 送信側のホストのIP 1111 > sendfile

```

解答欄(クライアント側)

```bash
# この下にコマンドを記述
nc -l 1111 < shadow

```
